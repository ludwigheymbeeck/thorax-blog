# A Blog Demo with Thorax

This is demo of my blog post at: http://thinkingonthinking.com/A-Blog-With-Thorax/ - A live demo is at: http://thorax-blog.herokuapp.com/#

To run:

```sh
$ npm install
$ bower install
$ node server.js
```

or

```sh
$ grunt
```
